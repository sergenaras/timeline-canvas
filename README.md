# Timeline Canvas

Bu proje, HTML Canvas kullanarak yüksek performanslı, zoom yapılabilir ve kaydırılabilir bir zaman çizelgesi bileşenidir. 

## Canlı Demo

Bu kütüphanenin canlı demosunu GitHub Pages üzerinde görüntüleyebilirsiniz.

https://sergenaras.github.io/timeline-canvas/

## Projenize Nasıl Eklersiniz?

Bu kütüphaneyi herhangi bir web projesine eklemenin en kolay yolu CDN (jsDelivr) kullanmaktır. Aşağıdaki adress üzerinden canlı kullanılabilir.

```sh
https://cdn.jsdelivr.net/gh/sergenaras/timeline-canvas@main/src/timeline.js
```
